# Changelog

All notable changes to the DSA Auto Sync extension will be documented here.

## [1.0.0] - 2026-04-11

### Added
- Initial release
- LeetCode submission detection
- GeeksforGeeks submission detection
- Automatic GitHub sync
- Toast notifications
- Stats display in popup
- Dashboard unlock after 5 problems
- GitHub OAuth integration
- Retry logic for API calls
- Local stats caching
- Debounce for duplicate submissions

### Features
- ✅ Lightweight vanilla JS implementation
- ✅ Manifest v3 compliance
- ✅ Efficient MutationObserver
- ✅ Smart error handling
- ✅ Clean, modern UI
- ✅ Performance optimized

### Technical
- Content script: Platform-specific extractors
- Background script: API handler with retry
- Popup: Stats display and OAuth flow
- Utils: Centralized API functions
- Styles: Minimal, responsive CSS

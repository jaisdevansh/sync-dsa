# 🚀 DSA Auto Sync - Extension Installation Guide

## For Users (Without Publishing to Chrome Web Store)

### Method 1: Download from GitHub

1. **Download the repository:**
   - Go to: https://github.com/jaisdevansh/sync-dsa
   - Click **Code** → **Download ZIP**
   - Extract the ZIP file

2. **Load extension in Chrome:**
   - Open Chrome browser
   - Go to `chrome://extensions/`
   - Enable **Developer mode** (toggle in top right)
   - Click **Load unpacked**
   - Navigate to extracted folder: `sync-dsa/apps/extension`
   - Select the folder and click **Select Folder**
   - Extension is now installed! ✅

### Method 2: Direct Extension Folder

If someone shared just the extension folder with you:

1. **Save the extension folder** anywhere on your computer
2. **Open Chrome** and go to `chrome://extensions/`
3. **Enable Developer mode** (top right toggle)
4. **Click "Load unpacked"**
5. **Select the extension folder**
6. Done! ✅

## 🎯 How to Use

1. **Click the extension icon** in Chrome toolbar
2. **Click "CONNECT TO GITHUB"**
3. **Authorize the app** on GitHub
4. **Solve problems** on LeetCode or GeeksforGeeks
5. **Check your GitHub repo** - solutions will be auto-synced!

---

## 🔄 Updating the Extension

When there's a new version:

1. Download the latest code from GitHub
2. Go to `chrome://extensions/`
3. Find **DSA Auto Sync** extension
4. Click the **Reload** button (circular arrow icon)
5. Extension updated! ✅

---

## 🐛 Troubleshooting

### Extension not showing up:
- Make sure you selected the correct folder (`apps/extension`)
- Check that Developer mode is enabled
- Try reloading the extension

### Not detecting submissions:
- Check that you're logged into LeetCode/GFG
- Open browser console (F12) and check for errors
- Reload the extension from `chrome://extensions/`

### GitHub not syncing:
- Make sure you clicked "CONNECT TO GITHUB"
- Check that you authorized the app
- Verify your GitHub token in extension popup
